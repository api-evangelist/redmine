<p>New maintenance releases for the Redmine 6.1, 6.0, and 5.1 series are now available to <a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Download">Download</a>. These releases address multiple security vulnerabilities along with various bug fixes and improvements.</p> <p><strong>Security Fixes</strong></p> All three versions (<a class="version" href="https://www.redmine.org/versions/220">6.1.2</a>, <a class="version" href="https://www.redmine.org/versions/221">6.0.9</a>, and <a class="version" href="https://www.redmine.org/versions/222">5.1.12</a>) include the following security fixes: <ul> <li>Defect #43661: Unsafe eval usage in AttachmentsHelper</li> <li>Defect #43690: Directory Traversal via Backslash-Separated Paths in Filesystem SCM</li> <li>Defect #43691: DOM (Stored) XSS in @mention autocomplete via unescaped user name</li> <li>Defect #43692: LDAP Injection (Unescaped Input in LDAP Search Filter)</li> <li>Defect #43694: DOM XSS: HTML Injection via Custom Field Name in Query Filter Generation</li> <li>Defect #43830: User who is allowed to view only their own time entries can retrieve other users’ time entry details by directly specifying the TimeEntry ID via the REST API</li> <li>Defect #43864 / #43840: Update Nokogiri to 1.18.9 (<a class="version" href="https://www.redmine.org/versions/222">5.1.12</a>) or 1.19.1 (<a class="version" href="https://www.redmine.org/versions/220">6.1.2</a> and <a class="version" href="https://www.redmine.org/versions/221">6.0.9</a>).</li> </ul> <p><strong>Maintenance Improvements</strong></p> Redmine <a class="version" href="https://www.redmine.org/versions/220">6.1.2</a> includes a significant number of maintenance fixes (30 in total). <ul> <li>A new series of fixes for RTL languages</li> <li>SVG Icons: Theme developers can now override the default icons sprite, please see <a class="issue tracker-2 status-5 priority-4 priority-default closed" title="Feature: Allow to change icons sprites from theme (Closed)" href="https://www.redmine.org/issues/43087">#43087</a> for details</li> <li><code>recent_pages</code> macro supports now <code>include_subprojects</code> parameter</li> </ul> <p><strong>Download and Changelog</strong><br />You can find the new versions in the <a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Download">Download</a> section.
