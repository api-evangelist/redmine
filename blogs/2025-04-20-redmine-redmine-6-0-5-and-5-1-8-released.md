<p>Maintenance releases <a class="version" href="https://www.redmine.org/versions/209">6.0.5</a> and <a class="version" href="https://www.redmine.org/versions/208">5.1.8</a> are now available to <a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Download">Download</a>, bringing a total of 32 bug fixes (<a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Changelog">Changelog</a>).</p> <p>Version <a class="version" href="https://www.redmine.org/versions/209">6.0.5</a> also continues the transition to modern SVG icons powered by Tabler, resolves an issue with plugin activity SVG icon paths when multiple plugins are loaded (<a class="issue tracker-1 status-5 priority-4 priority-default closed" title="Defect: Plugin activity icons broken when multiple plugins are loaded (Closed)" href="https://www.redmine.org/issues/42509">#42509</a>) and improves the support for RTL languages (<a class="issue tracker-1 status-5 priority-4 priority-default closed" title="Defect: Fix sidebar switch button display in RTL language (Closed)" href="https://www.redmine.org/issues/42575">#42575</a>, <a class="issue tracker-1 status-5 priority-4 priority-default closed" title="Defect: Improve SVG icon compatibility with RTL languages (Closed)" href="https://www.redmine.org/issues/42465">#42465</a>).</p> <p>Thanks everyone for their contributions.</p>
