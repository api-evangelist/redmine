<p>New maintenance releases for the Redmine 6.1, 6.0, and 5.1 series are now available to <a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Download">Download</a>. These releases address three security vulnerabilities along with various bug fixes and improvements.</p> <p><strong>Security Fixes</strong></p> All three versions (<a class="version" href="https://www.redmine.org/versions/218">6.1.1</a>, <a class="version" href="https://www.redmine.org/versions/217">6.0.8</a>, and <a class="version" href="https://www.redmine.org/versions/219">5.1.11</a>) include the following security fixes: <ul> <li>Defect #43451: PostScript disguised as PDF can lead to arbitrary file operations via thumbnail generation</li> <li>Defect #43634: Authorization bypass in Redmine allows modification of attachment metadata on invisible issues</li> <li>Defect #43635: Authorization bypass in Redmine allows deletion of attachment on invisible issues</li> </ul> <p><strong>Maintenance Improvements</strong></p> Redmine <a class="version" href="https://www.redmine.org/versions/218">6.1.1</a> includes a significant number of maintenance fixes (34 in total), with a particular focus on the user interface: <ul> <li>RTL Support: Numerous fixes for RTL layouts, including corrected positioning for reaction buttons, copy buttons, and avatars.</li> <li>Text Formatting: Improvements to CommonMark alerts, including localized titles (note, tip, warning, etc.), a new CJK-friendly emphasis extension and automatic list markers support for task list items (<a class="issue tracker-2 status-5 priority-4 priority-default closed" title="Feature: Enable CJK-friendly emphasis extension for CommonMark (Closed)" href="https://www.redmine.org/issues/43234">#43234</a>, <a class="issue tracker-3 status-5 priority-4 priority-default closed" title="Patch: Localize default commonmark alert titles (note, tip, warning, caution and important) (Closed)" href="https://www.redmine.org/issues/43379">#43379</a>, <a class="issue tracker-1 status-5 priority-4 priority-default closed" title="Defect: Automatic list marker does not work for task list items (Closed)" href="https://www.redmine.org/issues/43265">#43265</a>).</li> <li>SVG Icons: Continued refinement of the new SVG icon system and visual consistency.</li> </ul> <p><strong>Download and Changelog</strong><br />You can find the new versions in the <a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Download">Download</a> section. For a complete list of changes, please review the detailed <a class="wiki-page" href="https://www.redmine.org/projects/redmine/wiki/Changelog">Changelog</a> for each version.</p> <p>Many thanks to all the contributors who helped with these releases, especially those who responsibly reported the security issues (Elweth from YesWeHack and to Abor).</p> <p>Happy New Year!</p>
